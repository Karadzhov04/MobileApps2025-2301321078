# ExpenseMate

## Идея
ExpenseMate е мобилно приложение за управление на лични разходи.  
Целта е да улесни потребителя в следенето на ежедневните разходи, категоризацията им и визуализацията на статистики.

---

## Как работи
- Потребителят добавя нов разход чрез **FAB бутон**.
- Разходите се съхраняват локално в **Room база данни**.
- Данните се визуализират в списък.
- Unit tests and UI тестове с **Espresso** гарантират правилното функциониране на основните екрани.

---

## Архитектура
- **MVVM** (Model–View–ViewModel)
    - **Model:** Room база + Repository
    - **ViewModel:** Управлява логиката и предоставя данни към View
    - **View:** Activity/Fragment, които показват UI
- **LiveData/Flow** за автоматично обновяване на интерфейса

---

## Потребителски поток
1. Стартиране на приложението → показва се списък с разходи.
2. Натискане на **FAB бутон** → отваря се AddExpenseActivity.
3. Попълване на данни (сума, категория, описание) → запис в Room база.
4. Връщане към списъка → новият разход се визуализира веднага.
